# WorkBuddy 全场景教程目录

> 本目录先建立完整内容地图，再按优先级把每个主题写成带配图、可复制指令、风险提示和故障排查的独立教程。

## 目录设计原则

1. **按“我要完成什么工作”分类**，而不是按模型、Skill 或产品术语分类；
2. **先收集现成教程和官方资料，再独立研究、交叉核对和重写**；
3. 每篇正式教程至少包含：准备材料、操作步骤、可复制指令、结果验收、常见错误、撤销恢复、隐私安全、参考来源；
4. 面向非技术用户的教程原则上配 2—5 张图：实际界面截图、流程图、目录结构图或结果示意图；
5. 官方资料只能证明功能存在，不能替代真实场景验证；未经完整实测统一标记为 B 级来源核对；
6. 自动生成内容先进入 Draft PR，不直接合并。

## 状态说明

- ✅ 已发布：仓库中已有完整教程；
- 🟡 待完善：已有基础教程，需要按“大文件教程”的图文标准升级；
- 🔵 已有来源：已找到可研究的官方或社区教程，尚未形成本站文章；
- ⚪ 待研究：暂未找到足够可靠资料，需要自行设计和实测。

## 第一阶段：必须优先完成的 30 篇

### P0-A：新手能跑通

1. WorkBuddy 从下载安装到第一个任务；
2. Ask、Plan、Craft 三种模式；
3. 工作目录和文件权限；
4. 模型怎么选；
5. Skill 安装与安全检查；
6. 结果文件、变更记录和验收；
7. 任务卡住、文件打不开、登录失败等排障。

### P0-B：最高频办公任务

8. PDF 总结与页码核对；
9. 本地大文件分批处理；
10. 多文档对比与合并；
11. Excel 清洗、统计和图表；
12. 多表合并和自动报表；
13. PPT 从资料到成品；
14. Word 正式文档生成与修改；
15. 会议纪要和行动项；
16. 周报、日报和月报；
17. 批量重命名、分类和归档文件；
18. 外文视频转写、翻译和摘要。

### P0-C：增长和决策场景

19. 竞品调研；
20. 行业研究报告；
21. 用户反馈归类和需求提炼；
22. 市场数据搜集与可视化；
23. 小红书选题、正文和封面文案；
24. 公众号文章从资料到排版稿；
25. 短视频脚本、分镜和口播稿；
26. 产品资料拆成多平台营销内容。

### P0-D：自动化和跨设备

27. 创建安全的定时自动化；
28. 每日资讯简报；
29. 每周工作总结和数据简报；
30. 微信/企微/飞书/钉钉远程下任务。

## 教程生产顺序

```text
搜集官方与社区教程
        ↓
建立来源卡片和内容缺口
        ↓
设计本站更安全、更完整的流程
        ↓
制作截图或 SVG 示意图
        ↓
生成 B 级教程草稿
        ↓
内容检查与人工审核
        ↓
实际执行后升级为 A 级实测
        ↓
定期检查是否过期
```

## 信息源与更新

- 官方文档和官方更新日志：每周检查；
- 腾讯云开发者社区等公开教程：每周发现和去重；
- 高访问文章、用户 Issue 和搜索高频问题：进入 P0/P1 候选；
- 每月检查所有 B 级教程是否需要重新核对；
- 每季度抽查高流量教程并补充真实截图。
