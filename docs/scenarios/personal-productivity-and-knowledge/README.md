适合需要管理待办、邮件、资料、日程和长期知识积累的职场人。

## 其他场景

- 收件箱和下载目录整理；
- 会议、邮件和待办统一汇总；
- 工作日结束复盘；
- 每周学习计划；
- 书籍和课程资料整理；
- 个人模板库；
- 旅行和出差资料包；
- 重要文件到期提醒；
- 常用指令沉淀成 Skill；
- 历史任务查找和归档。

## 安全要求

- 邮件、日历和网盘授权使用最小权限；
- 自动发送、创建邀请和共享文件前确认；
- 不把所有私人文件长期开放给完全访问；
- 知识库回答需要显示来源和最后更新时间；
- 个人目标和工作优先级建议只能作为参考。

## 每篇教程需要的配图

- 待办从收集到完成的流程；
- 日历、邮件、网盘和本地文件关系图；
- 个人知识库目录结构；
- 每日/每周回顾模板；
- 授权、执行和人工确认边界。
