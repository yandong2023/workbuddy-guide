## P1：常见任务

- 从合同、发票、简历等文件中批量提取字段；
- 将图片中的表格恢复成 Excel；
- 给大量照片按日期、地点、项目分类；
- 批量压缩图片和文档；
- 文件夹内容生成目录和索引；
- 从历史资料中建立可搜索的知识库；
- 文档脱敏：姓名、电话、身份证号和客户信息；
- 多语言文档翻译并保持原格式；
- PDF 拆分、合并、旋转和页码整理；
- 电子书、培训资料和课程讲义整理。
