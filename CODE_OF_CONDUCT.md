# 社区行为准则

我们希望这个项目成为一个对普通用户、内容贡献者和技术贡献者都友好的中文社区。

## 我们鼓励

- 用清楚、尊重和可验证的方式讨论问题；
- 指出教程错误时提供页面、版本和复现信息；
- 对新用户解释概念，不嘲笑基础问题；
- 承认不确定性，不把猜测包装成事实；
- 尊重原创作者、来源和许可证；
- 在截图、日志和样例中主动保护隐私；
- 对不同职业、经验和使用方式保持包容。

## 不可接受的行为

- 人身攻击、骚扰、歧视和侮辱；
- 发布他人的个人信息、账号、文件或业务数据；
- 搬运付费内容、盗用图片或大段复制第三方教程；
- 伪造实测、截图、来源、产品版本或执行结果；
- 引导他人绕过账号、付款、安全和授权限制；
- 利用 Issue、PR 和评论发布广告、垃圾链接或恶意文件；
- 明知存在风险仍鼓励用户直接删除、覆盖或公开发送重要内容。

## 处理方式

维护者可以根据情况：

1. 提醒修改措辞或删除敏感内容；
2. 关闭跑题、重复、广告或不安全的 Issue/PR；
3. 删除包含隐私、密钥、恶意链接和侵权内容的评论；
4. 暂时或永久限制持续违反规则的参与者。

## 适用范围

本准则适用于仓库的 Issue、Pull Request、Review、Discussion，以及以本项目名义进行的公开交流。

参与本项目即表示同意遵守本准则。
