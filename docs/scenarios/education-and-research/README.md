# 教学、学习与研究

适合教师、培训人员、学生、研究人员和需要整理课程与专业资料的人。

## 其他场景

- 教案和课程大纲；
- 培训签到与反馈分析；
- 阅读笔记和知识卡片；
- 术语表和双语词汇；
- 课程作业评分标准草稿；
- 研究日志和实验记录整理；
- 调研问卷初稿；
- 学术会议材料整理；
- 科普文章和公众版本摘要；
- 教学资料多语言本地化。

## 诚信与安全要求

- 不替学生完成需要本人独立完成的作业或考试；
- 不伪造研究数据、引用、访谈、实验和参考文献；
- 摘要与结论应可追溯到原资料；
- 涉及受试者和学生数据时必须脱敏；
- 题目、答案和评分建议需要教师人工审核；
- 医疗、法律等专业培训内容应由专业人员复核。

## 每篇教程需要的配图

- 资料到讲义/PPT/练习的流程；
- 课程结构和学习目标图；
- 多篇材料对比矩阵；
- 研究资料库目录示意；
- “AI 辅助 → 人工学习/研究 → 引用核对”的边界图。
