## 推荐工作方式

1. 把原文件复制到独立工作目录；
2. 先让 WorkBuddy 检查输入和制定计划；
3. 明确输出格式和保存位置；
4. 不确定的信息标记“待确认”；
5. 不修改原文件；
6. 完成后检查数字、引用、文件格式和变更范围。

涉及客户、员工、患者、合同或经营数据时，应先脱敏，并遵守所在组织的数据安全要求。
