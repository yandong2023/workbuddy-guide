## 强制标准

- 业务指标必须说明数据口径；
- 不得自动下单、付款、修改库存或关闭异常；
- 供应商和采购决策需要人工审批；
- SOP 必须标注版本、适用范围和负责人；
- 批量文件或数据处理应保留原始记录和变更日志；
- 质量、安全和生产事故不能只依据 AI 判断。
