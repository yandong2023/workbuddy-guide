## 内容教程的强制标准

1. 不得虚构用户案例、效果数字和客户评价；
2. 引用数据必须保留来源；
3. 不自动发布，先生成草稿和预览；
4. 平台敏感词、广告法和版权风险需人工检查；
5. 图片、视频、音乐和第三方文章不得无授权搬运；
6. 同一素材跨平台改写时，应适配平台结构，而不是原文复制。
