## 安全要求

- 客户姓名、电话、邮箱、合同和订单必须最小化处理；
- 不得自动承诺价格、交期、赔偿、退款和产品能力；
- 对外消息默认只生成草稿，不直接发送；
- CRM 修改、客户标签和工单关闭前必须人工确认；
- 模型生成的客户画像和商机判断不得作为唯一决策依据。

## 每篇教程需要的配图

- 客户信息输入到跟进任务的流程图；
- 销售漏斗和阶段示意；
- 客户反馈分类矩阵；
- 工单分级规则图；
- “生成草稿 → 人工确认 → 对外发送”的安全流程。
