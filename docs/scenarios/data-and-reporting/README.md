## 数据教程的强制验收项

1. 原始文件是否保持不变；
2. 字段类型、日期、百分比和金额格式是否正确；
3. 核心指标能否抽样复算；
4. 数据清洗规则是否有记录；
5. 图表是否标明标题、单位、时间和口径；
6. 缺失或异常数据是否被隐瞒；
7. 最终结论是否能追溯到原始数据。
