## 高风险边界

- 招聘、绩效、晋升、解雇和薪酬决策不能只依赖模型；
- 简历、身份证、工资、银行账号和健康信息属于敏感数据；
- 财务数字必须与原始凭证和系统抽样核对；
- 报销、付款、采购、发薪和审批动作不得自动执行；
- 制度、合同和正式通知发布前应经过负责人或法务审批。
