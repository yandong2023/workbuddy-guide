## P1：可复制自动化

- 每日待办和日程摘要；
- 每周项目风险汇总；
- 每月客户反馈报告；
- 每日销售数据异常检查；
- 文件夹新增文件自动整理；
- 定时备份结果文件；
- 固定网页变化监控；
- 邮箱附件自动分类；
- 会议前资料准备；
- 内容日历提醒；
- 固定模板报表生成；
- 工作空间过期清理预览。

## 自动化的强制标准

1. 先手动执行成功 2—3 次，再设置定时；
2. 明确输入目录、文件名规则、时间范围和输出位置；
3. 输出文件名包含日期，避免覆盖；
4. 没有输入时应停止，而不是复用旧数据；
5. 发送、删除、发布、付款和账号修改必须人工确认；
6. 记录上次运行时间、结果和错误；
7. 防止同一任务重复执行和重复发送；
8. 提供暂停、停用和恢复方法。
