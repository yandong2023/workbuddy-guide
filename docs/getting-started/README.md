## 新手安全原则

- 使用单独测试目录；
- 原文件先复制再处理；
- 默认使用默认权限；
- 新任务先 Ask 或 Plan；
- 删除、覆盖、发送和发布前人工确认；
- 任务完成后检查实际文件和变更记录。
