## P1：研究任务

- 政策和法规资料整理；
- 招投标和采购信息研究；
- 学术或技术趋势简报；
- 公司背景与商业模式研究；
- 供应商对比和采购建议；
- 城市、地点和出差调研；
- 投资、融资和企业动态追踪；
- 客户行业研究；
- 搜索结果和新闻去重；
- 调研问题清单和访谈提纲；
- 假设、证据和反证矩阵；
- 决策备忘录。

## 调研教程的强制标准

1. 每个关键事实保留原始来源链接；
2. 标记发布日期、访问日期和数据年份；
3. 官方、一手资料优先，媒体和社区观点分开；
4. 多来源冲突时并列展示，不让模型擅自裁决；
5. 推断必须明确写成推断；
6. 不允许引用搜索摘要替代原文；
7. 对价格、功能、政策等易变化内容标记最后核对日期。

## 每篇教程需要的配图

- 竞品对比矩阵；
- 假设—证据—结论关系图；
- 自动监控和月度更新流程图。
