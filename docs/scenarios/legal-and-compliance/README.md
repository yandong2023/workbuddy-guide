# 法务、合同与合规

适合法务、采购、销售、行政、项目负责人和需要阅读合同、政策或制度的人。

> WorkBuddy 可以帮助整理、比较和标注材料，但不能替代律师、合规负责人或正式法律意见。

## 强制安全要求

1. 不把 AI 输出描述为法律意见；
2. 不自动签署、发送、接受或修改正式合同；
3. 风险结论必须附对应条款和页码；
4. 法律、政策和监管信息必须核对当前有效版本；
5. 合同中的个人信息、商业秘密和价格应脱敏；
6. 对关键条款必须明确标记“待法务确认”；
7. 不得自动推断未写入合同的权利义务。
