# WorkBuddy 是什么：它和普通 AI 聊天有什么不同

> 验证状态：B 级来源核对。本文依据官方文档整理，尚未完成本项目的完整人工实测。

WorkBuddy 是面向工作场景的桌面 AI Agent。它不只返回一段文字，还可以在用户授权范围内读取本地文件、拆解任务、生成文档、分析数据并交付文件结果。

## 它适合做什么

官方文档列出的典型场景包括：

- 生成工作报告、会议纪要和技术文档；
- 分析表格数据并生成可视化；
- 制作演示文稿和研究报告；
- 总结 PDF、整理文件和批量处理资料；
- 执行需要多个步骤的研究或办公任务。

## 它和普通聊天机器人的区别

普通聊天工具通常给出建议或文本。WorkBuddy 的重点是“执行”：它会规划步骤、操作授权目录中的文件，并输出可以检查的交付物。

这也意味着你需要关注两件事：

1. **任务描述是否清楚**：目标、输入、输出格式和限制越具体，结果越容易检查；
2. **权限是否合理**：不要为了少点几次确认，就长期开放不必要的完全访问权限。

## 不适合直接交给它的任务

未经人工确认，不建议直接让 Agent：

- 删除或覆盖唯一一份文件；
- 向客户、同事或公众自动发送内容；
- 处理包含密钥、财务账号或大量个人隐私的数据；
- 自动支付、下单或修改重要账号权限；
- 把未经核对的研究结果直接作为正式结论发布。
