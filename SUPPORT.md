# 获取帮助

## 先检查这些入口

- 第一次使用：[5 分钟开始](START-HERE.md)
- 查找具体任务：[全场景教程目录](docs/scenarios/README.md)
- 直接复制模板：[任务模板库](docs/prompt-library/README.md)
- 任务前后检查：[执行检查清单](docs/checklists/README.md)
- 常见故障：[故障解决目录](docs/troubleshooting/README.md)

## 在 Issue 中提问

选择最接近的问题模板，并尽量提供：

- 对应教程或任务；
- Windows 或 macOS 版本；
- WorkBuddy 版本；
- 使用的模式、模型、Skill 和连接器；
- 完整错误文本；
- 已尝试的步骤；
- 一个可以复现问题的脱敏样例。

问题描述越具体，越容易得到有效帮助。

## 不要公开提交

- API Key、Token、Cookie 和密码；
- OAuth 授权链接、二维码和回调参数；
- 客户、员工、患者、合同、订单和财务资料；
- 未脱敏截图、日志、录音和本地路径；
- 公司内部服务器、数据库和网络信息。

安全问题请先阅读 [`SECURITY.md`](SECURITY.md)。

## 本项目能提供什么

本项目可以帮助：

- 整理公开资料和官方文档；
- 提供可复制的安全任务模板；
- 解释教程和常见失败；
- 收集版本变化和实测反馈；
- 改进教程、配图和检查清单。

本项目不能替代：

- WorkBuddy 官方客服和账号支持；
- 法律、医疗、财务和人事专业判断；
- 公司 IT、安全和合规审批；
- 对文件恢复、付款、发布和外部发送的最终确认。
